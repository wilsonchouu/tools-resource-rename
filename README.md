# tools_resource_rename
切图文件分类小工具

### Jars (Just Click It !)

- drawable.jar
- mipmap.jar
- gui.jar (2017/8/21新增)

### How to use

- drawable.jar
- mipmap.jar

复制到资源文件夹并双击运行，资源文件即会自动分类为(drawable/mipmap-xhdpi)和(drawable/mipmap-xxhdpi)

- gui.jar
